Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML sets the HTML content inside an element, including all HTML tags.
innerText sets the visible text content of an element, excluding hidden elements.
textContent includes all text, even if hidden.

Question-3: What is event delegation in the DOM?

Event delegation is a way in JavaScript to handle events for many child elements by adding just one event listener to a parent element.
This is useful when i have lots of similar elements and want to make event handling faster and easier. Instead of adding a listener to each child,
the parent listens for events and handles them when they happen on any of its children.

Question-4: What is event bubbling in the DOM?

Event Bubbling is a concept in the DOM. It happens when an element receives an event, and that event bubbles up to its parent and ancestor elements in the DOM tree until it gets to the root element.
